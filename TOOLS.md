# Tools and Dependencies

## Required

- Git
- Python 3.10 or newer
- Shell terminal (macOS/Linux shell, Windows PowerShell or Git Bash)

## Optional

- VS Code, Cursor, or any Markdown editor
- `make` (if you later add convenience build commands)

## Verify Your Environment

```bash
git --version
python3 --version
```

Expected:
- Git prints a version string
- Python prints `3.10+`
