# Facilitator Lite Guide

Use this guide to run a simple 30-minute internal AI Fluency 101 session.

## Agenda

- 0-5 min: Why AI fluency matters
- 5-12 min: 4D model
- 12-20 min: Role-specific examples
- 20-26 min: Assessment walkthrough
- 26-30 min: 30-day plan commitment

## Prep

- Share `START-HERE.md` ahead of session
- Ask each attendee to choose one function track
- Ask each attendee to bring one recurring workflow example

## During Session

- Poll attendees on baseline confidence (1-4)
- Have each attendee define one 30-day outcome
- Pair attendees to review each other's first draft plan

## After Session

- Collect first draft assessments
- Track 30-day commitments
- Run a 15-minute check-in after 2 weeks
