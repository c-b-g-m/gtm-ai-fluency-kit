# Marketing AI Fluency 30/60/90 Plan

## Role Context

Audience: marketing individual contributors and managers.

Primary outcomes:
- Increase campaign quality and iteration speed
- Build repeatable AI-assisted campaign workflows
- Improve quality control and accountability practices

## 30 Days (Foundation)

- Document 3 recurring workflows (content drafting, campaign ideation, performance review).
- Build one reusable prompt pattern per workflow.
- Create quality checklist for brand voice, factual claims, and audience fit.
- Submit one decision log per week (keep/edit/discard).

Evidence:
- Prompt library v1
- 3 before/after output samples
- Weekly decision logs

## 60 Days (Operationalization)

- Standardize one AI-assisted campaign brief workflow.
- Add a review pass for hallucination/risk checks.
- Measure baseline vs current output cycle time and quality.
- Share one internal playbook page for team reuse.

Evidence:
- Campaign brief template
- Review checklist in active use
- Before/after metrics snapshot

## 90 Days (Scale)

- Implement a lightweight always-on workflow (for example, content idea mining or performance synthesis).
- Train 1-2 teammates on your workflow.
- Define accountability norms for publishing AI-assisted content.

Evidence:
- Workflow map v2
- Peer enablement notes
- Marketing diligence statement
