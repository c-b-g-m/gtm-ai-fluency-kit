# AI Fluency 101 Assessment Rubric (V1)

This rubric defines the 4D competency model with role-calibrated expectations by function.

## Scoring Model

- Scale: 1 to 3 per domain
  - 1 = Emerging
  - 2 = Developing
  - 3 = Capable
- Evidence requirement: score only when at least one observable artifact supports claim.

## Domain Criteria

## Delegation

### 1 (Emerging)
- Uses AI reactively for one-off tasks.
- No clear task decomposition or success criteria.

### 2 (Developing)
- Identifies recurring tasks where AI may help.
- Partial reasoning for tool and task choices.

### 3 (Capable)
- Uses repeatable delegation patterns for core workflows.
- Defines goals, constraints, and tradeoffs before execution.

## Description

### 1 (Emerging)
- Uses vague prompts with little context.
- Limited iteration when output quality is poor.

### 2 (Developing)
- Adds basic context and revises prompts occasionally.
- Some role-specific prompting patterns are visible.

### 3 (Capable)
- Builds strong context and iterates intentionally.
- Maintains reusable prompt structures for recurring work.

## Discernment

### 1 (Emerging)
- Accepts outputs with limited quality checks.
- Uses subjective evaluation only.

### 2 (Developing)
- Checks for obvious issues and revises some outputs.
- Uses partial quality criteria.

### 3 (Capable)
- Applies explicit standards and catches non-obvious issues.
- Documents decisions on what was kept, edited, or discarded.

## Diligence

### 1 (Emerging)
- Minimal disclosure or accountability framing.
- Inconsistent awareness of policy or privacy boundaries.

### 2 (Developing)
- Shows some transparency and risk awareness.
- Applies diligence checks in higher-risk situations.

### 3 (Capable)
- Consistently documents AI assistance and review responsibility.
- Applies policy, privacy, and verification checks by default.

## Required Evidence Artifacts

Use at least 3 of the following:
- Annotated chat logs
- Before/after output samples
- Decision log (keep/edit/discard rationale)
- Diligence statement (disclosure, risk checks, accountability)
- Workflow artifact (prompt library, checklist, automation map)

## Output Format

Assessment results should include:
1. Domain scores
2. Evidence used
3. Top 3 strengths
4. Top 3 gaps
5. 30-day action plan starter
